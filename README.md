# App del Tiempo - jjossuee008

Esta es una aplicación web interactiva desarrollada como proyecto de aprendizaje, que permite consultar el estado del clima de cualquier ciudad del mundo en tiempo real.

Utiliza la Fetch API para conectar con los servidores de OpenWeatherMap y obtener datos meteorológicos precisos.

---

## Características Principales

* **Identidad Visual Dinámica:** La aplicación cambia su fondo y estilo según el clima (Sol, Nubes, Lluvia, Niebla, etc.).
* **Glassmorphism Design:** Interfaz moderna con efectos de desenfoque y transparencias.
* **Búsqueda Dual:** Permite buscar haciendo clic en el botón o pulsando la tecla Enter.
* **Gestión de Errores:** Mensajes claros cuando una ciudad no es encontrada.

---

## Tecnologías Utilizadas

| Tecnología | Uso Principal |
| :--- | :--- |
| **HTML5** | Estructura semántica de la aplicación |
| **CSS3** | Diseño, efectos de cristal y animaciones |
| **JavaScript** | Lógica de la API, manipulación del DOM y eventos |
| **OpenWeather API** | Fuente de datos climáticos en tiempo real |
